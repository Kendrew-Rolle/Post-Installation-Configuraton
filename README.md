<h1>osTicket - Post-Install Configuration</h1>
This tutorial outlines the post-install configuration of the open-source help desk ticketing system osTicket.<br />


<h2>Video Demonstration</h2>

- ### [YouTube: How To Configure osTicket, post-installation](https://www.youtube.com)

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Operating Systems Used </h2>

- Windows 11 Pro

<h2>Post-Install Configuration Objectives</h2>

- Configure Roles
- Configure Departments
- Configure Agents
- Configure SLAs
- Configure Help Topics

<h2>Configuration Steps</h2>

<p>

<p>
-Within the admin panel interface, navigate to the agents tab.

-Click the Roles tab.

-In the Roles tab click Add New Role.

-Name said role.

-Navigate to Permissions.

-Add all necessary permissions relevant to that role.

  Here is a link to the video walkthrough on how to configure Roles in osTickets :(https://youtu.be/gdslJhAWqSU)
</p>
<br />

<p>
<img width="1178" height="433" alt="Screenshot 2025-08-10 130412" src="https://github.com/user-attachments/assets/4818a0c9-04fa-4189-9f14-6ac6d4970e60" />
</p>
<p>
-Within the admin panel interface, navigate to the agents tab.

-Click the Departments tab.

-In the Departments tab click Add New Department.

-Name said Department.

-Add any necessary or relevant information required.

-Click Create Department.
  
  Here is a link to the video walkthrough on how to configure Departments in osTickets :(https://youtu.be/CVu0RXaFhMc)
</p>
<br />

<p>
<img width="1179" height="423" alt="Screenshot 2025-08-10 130516" src="https://github.com/user-attachments/assets/253ae265-cf34-4d98-a7eb-63d1e2708a12" />
</p>
<p>
-Within the admin panel interface, navigate to the agents tab.

-In the Agents tab click Add New Agent.

-Add the name, email, and username of the Agent.

-Navigate to the Access tab.

-Add new agent to the necessary department and provide relevant access.

-Click create.
  
  Here is a link to the video walkthrough on how to configure Agents in osTickets :(https://youtu.be/3ClAt8SoTVM)
</p>
<br />

<p>
<img width="1176" height="375" alt="Screenshot 2025-08-10 130304" src="https://github.com/user-attachments/assets/1539d23a-4992-4b10-88f4-1b99b5fab6c9" />
</p>
<p>
-Within the admin panel interface, navigate to the Manage tab.

-Click the SLA tab.

-In the SLA tab click Add New SLA Plan.

-Name said SLA.

-Choose grace period for SLA (e.g., 8 hours).

-Choose Schedule for the SLA (e.g., 24/7).

-Click Add Plan.
  
Here is a link to the video walkthrough on how to configure SLAs in osTickets :(https://youtu.be/6D9_EdBMfzw)
</p>
<br />

<p>

<p>
-Within the admin panel interface, navigate to the Manage tab.

-Click the Help Topic tab.

-In the Help Topic tab click Add New Help Topic Plan.

-Name said Help Topic.

-(Optional) You can add a Parent Topic.

-(Optional) You can navigate to New Topic Options tab and further customize the topic.

-Click Add Topic.
  
  Here is a link to the video walkthrough on how to configure Help Topics in osTickets :(https://youtu.be/veqg1Nf9-lE)
</p>
